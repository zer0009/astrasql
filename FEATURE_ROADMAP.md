## ðŸ“¦ Distribution Model

**Important**: This project follows a closed-source distribution model:
- GitHub repository contains **executable files only** (EXE, DMG, AppImage)
- **No source code** is published on GitHub
- Installation is as simple as downloading and running the executable
- All updates are delivered through the auto-updater or new executable releases

### Installation Process
1. Download the executable from GitHub releases
2. Run the installer (Windows) or mount/open (macOS/Linux)
3. Follow the setup wizard
4. Connect your database
5. Start querying with natural language
